# WHITeN
is a chatbot for depressed people to talk to
# Running
use the run_desktop.bat for the desktop app and run_server.bat to run the local server.
# Requirements
Python >= 3.8
preferably 3.10
rivescript >= 1.15.0
wikipedia >= 1.4.0
google >= 3.0.0
pyowm >= 3.2.0
PySide6 >= 6.2.1
markdown >= 3.3.4
jokeapi >= 1.0.2
